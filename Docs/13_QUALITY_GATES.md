# Quality Gates

## Reject Immediately
- obvious hand errors,
- incorrect limb proportions,
- duplicated body parts,
- unreadable pose,
- geometric clothing artifacts,
- severe style drift,
- accidental text,
- excessive particulate density,
- excessive micro-texturing,
- environment overwhelmed by manifestation effects,
- generic anime rendering when painterly rendering is required.

## Review Before Approval
- anatomy,
- composition,
- silhouette,
- lighting,
- value grouping,
- color balance,
- edge hierarchy,
- focal clarity,
- pose variety,
- environmental identity,
- manifestation logic,
- narrative usefulness.

## Texture Gate
Reject images where surface texture becomes the primary visual event.
