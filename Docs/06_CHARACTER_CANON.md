# Character Canon

## Sol

### Child Sol
- Nine years old in the opening era.
- Frail, uncertain, frightened, and amnesiac upon arrival.
- Separate visual reference from Adult Sol.

### Adult Sol
- Mature and grown-in rather than youthful-anime.
- Pale or gray-white eyes.
- Dark hair with auburn, oxblood, or red undertones, especially visible in light.
- Pendant is a recurring identifier.
- Current approved adult reference remains canonical until explicitly replaced.

### Sol's Manifestation
- Red-orange palette with silver streaks.
- Smoke is the dominant medium identity.
- Painterly and unstable rather than geometric.
- Scythe resembles smoke given form, not a rigid construct.
- Coat and manifestation-side arm may become shadowed or smoke-like when active.

## Kael
- White hair.
- Purple eyes.
- Purple accents in clothing or uniform.
- Pensive visual presence.
- Current approved reference remains canonical.

## Vryn
- Confident, relaxed, self-aware.
- Current approved visual reference remains canonical.
- Preserve increased color variety.

## Auric
- Mentor character.
- Canonical name: Auric.
- Current approved mentor reference remains canonical.

## Supporting Characters
Create strong distinguishing packages rather than automatically training a LoRA for every minor character.
Use:
- reference sheet,
- expression sheet,
- wardrobe,
- silhouette notes,
- palette,
- prompt snippet,
- optional LoRA only when repeated consistency demands it.
